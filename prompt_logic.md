Objective

The objective of this prompt system is to generate SEO-optimized, business-focused blog content for a local fitness gym website.
The system is designed to improve search engine visibility, attract local traffic, and support content marketing goals.

 1. SEO Context Prompt Logic
The SEO context prompt defines:
Business type (Fitness Gym)
Location (Hyderabad)
Target audience (beginners and working professionals)
Content tone (helpful, simple, professional)
Why this matters:
Including location and audience ensures the content is optimized for local SEO searches and avoids generic fitness blog content.

 3. Blog Topic Generation Prompt Logic
The topic generation prompt focuses on:
Common fitness problems
Beginner-friendly topics
Local search intent
Why this matters:
SEO success depends on choosing the right topics. This prompt ensures blogs target real user queries that people in Hyderabad are likely to search for.

 5. Blog Outline Prompt Logic
The outline prompt generates:
Proper H1, H2, and H3 structure
Logical content flow
FAQ section

Why this matters:

A clear structure improves readability, SEO ranking, and helps search engines understand the content hierarchy.

 4. SEO Blog Content Prompt Logic
The blog content prompt instructs the AI to:
Follow the generated outline
Use local keywords naturally
Keep explanations simple and helpful
Add a soft call-to-action
Why this matters:
This ensures the blog is not only optimized for search engines but also valuable for real readers, increasing engagement and conversions.

 6. Content Cluster Prompt Logic
The content cluster prompt creates:
One pillar blog topic
Multiple supporting blog topics
Internal linking suggestions
Why this matters:
Content clusters help search engines identify topic authority. Internal linking improves SEO strength and keeps users on the website longer.
